### 5. –§—É–Ω–∫—Ü–∏—è `escapeHtml()`

**–§–∞–π–ª:** `frontend/index.html` (—Å—Ç—Ä–æ–∫–∞ ~835)

 * –£—Ç–∏–ª–∏—Ç–∞ –¥–ª—è —ç–∫—Ä–∞–Ω–∏—Ä–æ–≤–∞–Ω–∏—è HTML
 */
function escapeHtml(text) {
    const map = {
        '&': '&amp;',
        '<': '&lt;',
        '>': '&gt;',
        '"': '&quot;',
        "'": '&#039;'
