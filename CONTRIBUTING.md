# Contributing to PANDORA

Спасибо за интерес к развитию PANDORA! Вот как вы можете помочь:

## Как начать

1. Fork репозиторий
2. Создайте feature branch (`git checkout -b feature/AmazingFeature`)
3. Commit изменения (`git commit -m 'Add some AmazingFeature'`)
4. Push в branch (`git push origin feature/AmazingFeature`)
5. Откройте Pull Request

## Стандарты кода

- Используйте типизацию в Python (type hints)
- Используйте TypeScript в frontend
- Следуйте PEP 8 для Python
- Используйте Prettier для JavaScript/TypeScript

## Процесс

1. **Перед запуском**: Обновитесь с main ветки
2. **Во время разработки**: Тестируйте часто
3. **Перед PR**: Запустите тесты и lint

## Благодарности

Спасибо всем, кто помогает развивать проект!
